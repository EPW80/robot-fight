# Robot-Fight.

## Purpose
The purpose of this program was to develop a simple robot combat game using the concepts tha we have learned up to this point. The player 
will attack their opponent reducing HP. In return the opponent will be able to attack the player and also reduce the players HP. The first 
to get to 0HP loses.
